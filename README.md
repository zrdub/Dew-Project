# Draw Ur Dew

Draw Ur Dew is an interactive, camera-based website with an iOS-style liquid glass concept. This website allows users to create digital dew effects and draw on the screen using real-time finger gestures.

## Key Features

- Modern frontend with liquid glass design
- Homepage with a dewy glass frame
- "have fun :)" text effect that resembles fingerprints on glass
- Real-time camera access
- Face detection using MediaPipe FaceMesh
- Open mouth detection to create a dew effect
- Index finger detection using MediaPipe Hands
- Draw or erase dew with finger gestures
- Dew effect using HTML Canvas
- Responsive for desktop and mobile

## Technologies Used

- React
- TypeScript
- Vite
- MediaPipe FaceMesh
- MediaPipe Hands
- HTML Canvas
- Custom CSS
